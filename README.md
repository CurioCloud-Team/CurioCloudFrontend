# CirioCloud 前端

## 概述
课研云 CurioCloud 旨在打造一款AI驱动的教师备课辅助系统，通过集成先进的人工智能技术，解决传统备课方式效率低、创新性不足的痛点。项目旨在实现教学设计的自动化、教学资源的智能化生成与个性化推荐、以及深度的学情分析，从而革命性地提升教师的备课效率与教学质量。该项目使用 Vue 3 + TypeScript + Vite，Tailwind CSS + daisyUI。

## 系统要求
* Node.js 20 或更高
* NPM 10 或更高

## 安装
1. 克隆这个仓库
2. `npm install`

## 构建
1. `npm run build`

### 运行站点
1. `npm run dev`
